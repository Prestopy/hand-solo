# Hand Solo
Hand Solo is a robotic arm project created by Bradley Chen, Aaron Lee, and me. This codebase is the program for the robotic arm, which has 3 degrees of freedom.

Here's our engineering journal: https://docs.google.com/presentation/d/1TmfcpO1rnT7csAdlobxXxSzgrCS0se2AeBmF0CUkoPE/edit?slide=id.g3b71aa7a6ad_2_6#slide=id.g3b71aa7a6ad_2_6

## Dev note
Please note that this project uses the WPILib FRC robot structure. So, in order for the code to run, please use the WPILib distribution of VScode.
